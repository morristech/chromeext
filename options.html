Verified.ninja Chrome Plugin

Chrome plugin last updated: 9/4/2015

What is Verified.ninja?

Verified.ninja allows people on dating websites to use our team to verify appearances. We make sure everyone on our website actually looks like their photos on dating websites. Once we verify multiple photos, we give our Verified.ninja status which makes sorting through dating profile photos much easier for you.

What does this Verified.ninja Chrome plugin do?

This plugin will automatically scan the pages of the dating website you are visiting and tell you which users are a Verified.ninja by adding the green Verified.ninja icon next to each username! This plugin will also add a red icon next to their users that are NOT a Verified.ninja.

What sites does this Chrome plugin work on?

We currently support these dating websites:

  • OKCupid
  • ChristianMingle
